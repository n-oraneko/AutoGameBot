      python3.10 <br>
      pytorch2.0　※cpuで運用想定
      <br>

 　■　pcにプロコンを接続<br>
 　■　main.py --trainingを起動し、ゲームを遊ぶ<br>
 　■　training_model.pyを実行しモデルの作成<br>
 　■　main.pyを実行しAIによるゲームの自動化を行う<br>
